# Personal blog (minimal)

Single-page, markdown-first personal site inspired by [Based Cooking](https://based.cooking/), [Luke Smith](https://lukesmith.xyz/), and Bear Blog.

## Edit content

- Edit `content.md`
- Refresh the page

## Run locally

Serve this folder (don’t open `index.html` directly, since the browser must fetch `content.md`).

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
